# Hearing aid effectiveness and dementia risk: a quasi-experimental analysis of seven cohort studies across 26 countries

This repository contains R code used for replication. All statistical analyses were performed with R version 4.3.2.

The public release data used in this paper included CHARLS, KLOSA, ELSA, SHARE, TILDA, HRS and MHAS, which will be available for download from their official websites: https://charls.pku.edu.cn/en/(CHARLS), https://survey.keis.or.kr/eng/myinfo/login.jsp (KLOSA), https://www.elsa-project.ac.uk/accessing-elsa-data (ELSA), https://share-eric.eu/data/data-access (SHARE), https://tilda.tcd.ie/data-workshop/index.php (TILDA), https://hrsdata.isr.umich.edu/data-products (HRS), https://mhasweb.org/DataProducts/Home.aspx (MHAS).

The code is structured with two sections.
1. Hearing aid effectiveness and dementia risk: a quasi-experimental analysis of seven cohort studies across 26 countries_1
2. Hearing aid effectiveness and dementia risk: a quasi-experimental analysis of seven cohort studies across 26 countries_2
